# Hello World
[*View on Codewars*](https://www.codewars.com/kata/hello-world)

Given an argument that may be a String or a Number, return a simple greeting.
# Example
- `hello('Noclip') => 'Hello Noclip'`
- `hello(13) => 'Hello 13'`
- `hello('there') => 'Hello there'`
